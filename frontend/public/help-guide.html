FarmSense AI — Help Guide

Register, connect the frontend to the backend, and confirm your data is saved in PostgreSQL.

1. Start services

cd farmsense-ai && docker compose up -d
cd frontend && npm run dev

Open http://localhost:5173. Ensure VITE_USE_MOCK_API=false in frontend/.env.

2. Register & sign in

Anyone can create an account. Your name, email, farm region, and size are stored in the database.

3. Add soil data (Plan)

Submit N, P, K, pH, and location. The API runs the Random Forest model and saves a recommendation run.

4. Check Settings

  • System status — API, PostgreSQL, ML model health
  • Saved data — soil reading count, recommendation runs, last update dates

5. Return later

Yes — sign in days or weeks later and your profile and crop history are still there.

Settings → Saved data shows counts from the server. Dashboard loads from GET /dashboard.

Health endpoints

  • GET http://localhost:8000/health
  • GET http://localhost:8000/health/ready (database + ML)
  • GET http://localhost:8000/auth/me (requires login token)
